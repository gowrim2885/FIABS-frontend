# AI Fiscal Intelligence Dashboard (FIABS)

A comprehensive, interactive dashboard for fiscal intelligence, forecasting, risk simulation, and decision support using React, Tailwind CSS, and Chart.js.

## Features

- 📊 **Budget Forecasting** - AI-powered fiscal predictions
- 🎯 **Risk Simulation** - Scenario analysis and risk modeling
- 💰 **Resource Allocation** - Optimize budget distribution
- 🔍 **Anomaly Detection** - Identify unusual fiscal patterns
- 🤖 **Decision Support System (DSS)** - AI recommendations
- 📈 **Interactive Charts** - Real-time data visualization
- 🎨 **Modern UI** - Built with Tailwind CSS

## Tech Stack

- **Frontend**: React 18
- **Styling**: Tailwind CSS
- **Charts**: Chart.js + react-chartjs-2
- **Routing**: React Router v6
- **Backend**: FastAPI/Python (to be integrated)

## Installation

```bash
npm install
npm start
```

```
src/

```
